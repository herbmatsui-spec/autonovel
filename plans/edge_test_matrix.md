# 尖り保全テストデータマトリクス

## 判定基準 (Oracle)
- **PASS**: 判定ロジックが `True` を返し、内容が保持されていると認めた場合。
- **FAIL**: 判定ロジックが `False` を返し、適切に「削られました」等の警告を出した場合。
