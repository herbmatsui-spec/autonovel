## 概要

- **全体カバー率**: 29%
- **総行数**: 13,848

## 詳細
- `src` および `streamlit_app` 全体を対象に測定。

- 多くのビジネスロジックおよびUIコンポーネントが未カバー状態にある。

- 特に `src.core.observability` や `src.backend.sanitizer` 周辺の不整合が見られる。
- これらは Phase 2-4 のリファクタリングおよび Phase 6 のテスト強化で対処する。
