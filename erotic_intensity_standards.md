# 官能の強度基準 (Erotic Intensity Standards)

本ドキュメントは、作品における官能表現の強度を定義し、セーフティ範囲および表現の方向性を管理するための基準である。

| 強度レベル | 定義 | 表現の主眼 | 具体的な描写範囲 | セーフティ判定 |
| :--- | :--- | :--- | :--- | :--- |
| **Lv.1: 暗示的** | 心理的な緊張感と雰囲気の醸成 | 視線、溜息、空気感、心理描写 | 接触は限定的（手をつなぐ、肩に触れる等）。情事の直前や直後を暗示。 | 完全セーフ |
| **Lv.2: 情緒的** | 身体的な惹きつけと親密さの強調 | 体温、心拍、肌の質感、軽い接触 | キス、抱擁、衣服越し、あるいは一部の肌の露出。感触の記述を中心とする。 | セーフ |
| **Lv.3: 官能的** | 身体的快楽と熱情の直接的な描写 | 吐息、喘ぎ、密着、身体的な反応 | 衣服を脱ぐ過程、愛撫、直接的な肌の接触。情事のプロセスを美的に描写。 | セーフティ上限 |
| **Lv.4: 直接的** | 具体的かつ詳細な行為の描写 | 性器の直接的言及、具体的な結合描写 | 行為の詳細なメカニズム、露骨な身体部位の名称使用。 | 制限/NG (規約抵触) |
| **Lv.5: 過激** | 極限の感覚変容と精神的溶解 | 身体境界の消失、理性の崩壊、快楽の飽和 | 物理的描写を超えた感覚的な「溶解」や「洪水」のような比喩表現。自己喪失を伴う極限的な快楽。 | セルフホスト専用 ( adult_selfhost ) |

## 運用ガイドライン

1. **セーフティ上限 (Lv.3) の維持**: 
   - 「何をしたか」という物理的な事実に集中せず、「どう感じたか」という主観的な快楽や情緒に変換して描写することで、強度をLv.3に保ちつつ官能性を高める。
2. **比喩の活用**:
   - 直接的な語彙を避け、比喩（例：「波のように」「溶け合う」）を用いることで、読者の想像力を刺激し、Lv.4に至ることなく官能性を演出する。
3. **構成の重要性**:
   - `Build` $\rightarrow$ `Peak` $\rightarrow$ `Afterglow` のフローを重視し、身体的なピークだけでなく、その前後の情緒的な充足感を丁寧に描写する。

4. **強度Lv.5 (Intense) の特別運用**:
  - `adult_selfhost` プリセットなどの制限のない環境においてのみ有効とする。
  - 物理的な露骨さ（Lv.4）ではなく、**感覚的な過激さ（感覚の変容・精神の崩壊）**に重点を置いた `intense` ボキャブラリを用いることで、品位を保ちつつ最大限の官能性を追求する。
